 Cjob_launcherPerforms the combined operations a queue scheduler like PBS or MOAB and a job launcher like SLURM (srun) or ALPS (aprun). The job launcher allocates nodes to each requested MPI job (or other application). Once nodes are allocated, the job_launcher has to assign MPI ranks to each node (mapping or indexing). Each application can request a specific allocation or indexing. However, it is ultimately the responsibility of the job launcher to decide on the final allocation/indexing. In most cases, the job_launcher will honor exactly each applications's request unless there is a conflict - in which case the job_launcher must arbitrate conflicting requests
