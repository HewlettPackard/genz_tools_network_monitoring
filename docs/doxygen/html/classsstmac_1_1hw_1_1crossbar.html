Detailed Description

The crossbar network generates a network which connects all nodes with only two hops: those to and from the crossbar.

Member Function Documentation

int sstmac::hw::crossbar::num_leaf_switches ( ) const
inlineoverridevirtual

Structured topologies can be direct (torus) or indirect (fat tree). We therefore need to distinguish the total number of switches and the number of leaf switches - i.e. those directly connected to nodes. For direct topologies, num_switches and num_leaf_switches are the same. For indirect, num_leaf_switches < num_switches.

Returns
The number of leaf switches directly connected to compute nodes
