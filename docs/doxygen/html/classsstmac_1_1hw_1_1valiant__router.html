Detailed Description

The valiant_router class Enacpsulates a router that performs oblivious, global routing according to "A Scheme for Fast Parallel Computation" by Valiant.

virtual int sstmac::hw::valiant_router::first_stage_vc ( int  topology_vc)
inlinevirtual

The topology object specifies a virtual channel based purely on geometry. However, the final virtual channel depends on both geometry and routing algorithm. In this case, we need a separate set of virtual channels depending on whether we are in the first stage (routing to the intermediate switch) or the second stage (routing to the final switch).
