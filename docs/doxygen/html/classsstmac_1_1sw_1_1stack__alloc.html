Detailed Description

A management type to handle dividing mmap-ed memory for use as ucontext stack(s). This is basically a very simple malloc which allocates uniform-size chunks (with the NX bit unset) and sets guard pages on each side of the allocated stacks.

This allocator does not return memory to the system until it is deleted, but regions can be allocated and free-d repeatedly.
