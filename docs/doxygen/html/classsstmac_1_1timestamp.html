Detailed Description

A basic container for time (subject to future transplant). Stores time as an integral number of picoseconds (tentatively). With 1 psec resolution, a 64 bit int can hold roughly +/- 106 days.

Intended to be reasonably compatible with ns3::HighPrecision time.
